# 🦎 Desafío 3 - Iguana Page

Proyecto desarrollado como parte del **Desafío 3 - Iguana Page**, enfocado en la creación de una **landing page informativa** utilizando **HTML5 y CSS3**.

El objetivo principal fue replicar un diseño propuesto, aplicando técnicas de maquetación web, estilos visuales, organización de contenido y uso de imágenes para construir una página moderna y atractiva.

## ⚙️ Cómo Ejecutarlo

1. Clona este repositorio:

```bash
git clone https://github.com/jaleival/desafio-3-iguana-page.git
```

2. Ingresa al proyecto:

```bash
cd desafio-3-iguana-page
```

3. Abre el archivo principal:

```bash
index.html
```

---

## 🌐 Publicación en GitHub Pages

1. Ir a **Settings**
2. Seleccionar **Pages**
3. Branch: `main`
4. Folder: `/root`
5. Guardar cambios
